节点服务器配置

  1. API 服务节点 同步块，广播事务，并提供链上基础数据的查询功能

    • 最低要求
      • CPU: 1 cores
      • RAM: 2GB
      • Storage SSD: 200GB
      • 带宽: 5Mbps
    • 推荐配置:
      • CPU: 4 cores
      • RAM: 8GB
      • Storage SSD: 200GB
      • 带宽: 5Mbps

  2. 验证者节点 出块，生产快

    • 最低配置:
      • CPU: 4 cores
      • RAM: 8GB
      • Storage SSD: 200GB
      • 带宽： 5Mbps
    • 推荐配置:
      • CPU: 8 cores
      • RAM: 32GB
      • Storage SSD: 500GB
      • 带宽: 20Mbps(越大越好)
  3. 全节点,种子节点
    • CPU: 4 cores
    • RAM: 8GB
    • Storage SSD: 200GB
    • 带宽： 4Mbps
